## 🚀 Setup

**1. Clone o repositório**

```bash
git clone https://github.com/lucassamuelDEV/codelandia-desafio-02.git
```

**2. Mude para o diretório do projeto**

```bash
cd codelandia-desafio-02/
```

**3. Instale as dependências do projeto**

```bash
yarn install
```

**4. Inicie o servidor**

```bash
yarn dev
```
